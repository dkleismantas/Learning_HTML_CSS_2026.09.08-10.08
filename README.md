# Learning_HTML_CSS_2026.09.08-10.08
In this module, I learned the basics of HTML and CSS. Here you can find the assignments and the work I completed for them.

# Technologies used
HTML, CSS, Visual Studio Code

# How to run the assignments
- Clone the repository.
- Open the folder of the subject you want to explore — inside you will find several assignments, numbered in order.
- In some cases, you can find the assignment requirements inside the Task folder of that specific assignment.
- Open the folder of the assignment you want to run in Visual Studio Code.
- From the visible files, select index.html.
- Install the Live Server extension in VS Code and click Go Live in the bottom‑right corner.
- Your browser will automatically open and display the result of the assignment.
